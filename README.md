# Memory Fragmentation Analyzer

A web-based simulation tool that demonstrates how dynamic memory allocation works in Operating Systems. This project visually represents internal and external fragmentation using different allocation strategies.

## Features

- 🔹 First Fit, Best Fit, Worst Fit, Next Fit algorithms
- 🔹 Real-time memory block visualization (128 blocks)
- 🔹 Internal & External Fragmentation calculation
- 🔹 Memory utilization tracking
- 🔹 Auto Simulation mode (random allocation)
- 🔹 Interactive dashboard controls

## Tech Stack

- React.js
- JavaScript
- CSS Grid
- SVG

## How to Run

npm install  
npm start  

## 🎯 Objective

To help students understand memory fragmentation visually.
